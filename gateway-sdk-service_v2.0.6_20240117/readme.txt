1. restart.sh is the running script for Linux systems
2. start.bat and stop.bat are running scripts for Windows systems
3. Before starting the Windows system, modify the corresponding java jdk1.8 installation path
start.bat -> Modify the path of JAVA_EXE=D:\"Program Files"\Java\jre8\bin\java
4. Before starting the Linux system, please configure the JAVA_HOME environment variable
