1. Document description
This document is mainly used for the face machine middleware HTTP interface
docking, suitable for system development and maintenance personnel.
1.1 Server preparation
1. Port 10010,10011,8190 is required
2. Support for windows and Linux systems
3. Install Java, run environment JDK 1.8
1.2 Service startup
1. windows System needs to modify the start t. The jdk path of the bat file,
like: set JAVA _ EXE = D: "Program Files" \ Java \ jre 8 \ bin \ java


--- PAGE 2 ---
2. Double click to run start. Bat, the popup below indicates that the middleware
runs successfully, close the popup middleware service is closed
1.3 Cloud configuration of devices
Before communicating with the cloud, the server should be configured in the Web
interface of the face device.
Note: The web configuration must first run the face recognition App.

--- PAGE 3 ---
1. Click the setting icon in the upper right corner of the device, enter the
password background and click Login to enter the Settings page
1. Click to enter the Communication Settings to configure the server
information

´üČ If the call interface returns the parameter exception, follow these steps:
´éí Check whether the parameter name is misspelled and whether there is

´éí Check whether the parameter value conforms to the specification, such as
